## About

Researcher in Machine Learning with a focus on real-world industrial applications.  
My work lies at the intersection of:

- Time Series Classification  
- Explainable Artificial Intelligence 
- Fault Detection in Chemical Processes  

I investigate data-driven and interpretable approaches for robust decision-making in non-stationary environments.
